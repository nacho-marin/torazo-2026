# Torazo 2026 🏔️

Guía visual del viaje familiar a la **Hostería de Torazo** (Cabranes, Asturias), del **5 al 9 de julio de 2026**. Para Sara, Laura y papá.

## 🌐 Ver la guía

**https://nacho-marin.github.io/torazo-2026/**

Página pública, se abre en cualquier móvil sin necesidad de login. Diseñada para consultarla durante los cuatro días del viaje.

## Qué incluye

- **Itinerario día a día** (dom 5 → jue 9 julio)
- **Tres rutas de senderismo circulares**: Molinos de Bimenes, Molinos del Río Profundu (Villaviciosa) y el Camín Encantáu (Ardisana, Llanes)
- **Spa Les Agües** de la hostería
- **Tabla de tiempos de coche** desde Torazo
- **Plan B** (Parque Principado) para días de orbayu
- **Checklist** de reservas y maleta
- **Teléfonos y enlaces** del hotel

## Cómo editar

El sitio es un único fichero `index.html` autocontenido (CSS inline, sin dependencias externas). Para actualizar la guía:

```bash
# editar index.html
git add -A
git commit -m "docs: <qué cambió>"
git push
```

GitHub Pages reconstruye la web en ~1 min sobre la misma URL.
